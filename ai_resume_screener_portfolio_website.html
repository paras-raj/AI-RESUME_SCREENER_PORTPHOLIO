AI Resume Screener

Smart & Swift Hiring – Matching Resumes with Job Descriptions using AI

🔍 Project Overview

An intelligent resume screening tool powered by Natural Language Processing. This project empowers recruiters by automatically analyzing resumes and job descriptions to find the best talent fit – fast, accurate, and efficient!

✨ Key Features

  • 📄 Parses PDF resumes with high accuracy
  • 🧠 Extracts keywords from job descriptions
  • 🎯 Calculates skill match percentage
  • 🚀 Express.js-powered REST API
  • 💻 Ready to plug into CLI or Frontend

🛠️ Technology Stack

Backend: Node.js, Express.js, Multer
Libraries: pdf-parse, fs, custom NLP
Frontend: HTML/CSS or React (optional)
Deployment: Render, Vercel, or Localhost
